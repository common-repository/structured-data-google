== Description ==

Add Google Structured data to your website, local map, opening hours, contact information with json data only avialble for google.

== Installation ==

1. Upload plugin folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
